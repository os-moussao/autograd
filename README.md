# Autograd
Autograd is an automatic differentiation library that works with Numpy arrays and operations.

It allows you to compute the gradients of an output (loss function) with respect to all its inputs (model parameters), which is essential for building and training machine learning models.

I built this library to understand how auto gradient and backpropagation work, and used it as the foundation engine to build my [micro-torch](https://github.com/os-moussao/micro-torch) framework.
